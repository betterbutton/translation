Raushan Raj sepsal stručný návod, který vám krok za krokem ukáže, jak hostovat statický web na GitHubu. Předpokládá, že již umíte nahrát kód na GitHub (třeba přes webové rozhraní), ale pokrývá postup od založení složek až po úspěšné zveřejnění.

Nejprve projekt nahrajete na GitHub, nastavíte soubor index.html do hlavní složky a povolíte GitHub Pages v nastavení. Po krátké chvíli bude web přístupný přes odkaz, který GitHub vygeneruje. Tato metoda nabízí snadný způsob publikace pro ty, kteří hledají rychlé řešení bez nákladů.
