Microsoft uvedl GenAIScript. Je to nástroj pro práci s velkými jazykovými modely (LLM) přímo v JavaScriptu, který umožňuje snadné vytváření a sdílení promptů. Nabízí integraci s Visual Studio Code a širokou podporu pro práci se soubory a daty, včetně nástrojů pro strukturovaný výstup, konverzi textů z PDF, prohledávání tabulek, a dokonce i spuštění LLM agentů a nástrojů v kontejnerech. GenAIScript podporuje také GitHub Models, LocalAI a bezpečnostní prvky pro obsah. Rozšiřuje možnosti pro vývojáře, aby mohli efektivně automatizovat a optimalizovat úkoly za využití nejmodernějších modelů AI.
